# shelf | RSS_Filter

### A text-based RSS feed filter

## Quickstart

```shell
git clone https://github.com/projects-shelf/RSS_Filter.git
cd RSS_Filter
docker-compose up -d
```

## Features

### Filters out RSS feed items containing links to domains listed in blocklist.txt.

Example `blocklist.txt`:

```txt
example.com
example.net
example.org
```
